# 견적 UI Implementation Plan (Plan 2/6)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development. Steps use checkbox (`- [ ]`) syntax.

**Goal:** 비개발자가 "집 짓기" 비유 화면에서 선택하면 실시간으로 예상 예산대(또는 상담필요)와 포함/별도가 뜨고, 연락처를 제출해 `/api/quotes`로 리드를 남기고 공유링크를 받는 화면을 만든다.

**Architecture:** Next.js App Router 클라이언트 컴포넌트가 선택 상태를 들고, **이미 있는 순수 `classify()`(lib/quote)를 클라에서 직접 호출해 실시간 미리보기**를 그린다. 제출은 Plan 1의 `POST /api/quotes`(서버가 재계산해 저장)로 보낸다 — 클라 미리보기와 서버 저장값은 같은 모듈을 쓰므로 일치한다. 디자인은 DESIGN.md 토큰(라이트/다크, accent #0066cc, 정제 미니멀+절제된 터미널 모티프).

**Tech Stack:** Next.js 16 · TypeScript · CSS Modules + CSS 변수(DESIGN.md) · Vitest + jsdom + @testing-library/react

**참조:** [DESIGN.md](../../../DESIGN.md) · [docs/design.md](../../design.md) 인터랙션 상태표·반응형/a11y · 명세서 §5(입력)·§7(포함/별도) · Plan 1 `src/lib/quote/*`, `POST /api/quotes`
