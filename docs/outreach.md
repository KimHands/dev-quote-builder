# 아웃리치 키트 (The Assignment)

> 이 계획 전체의 **1순위 과제.** 사이트 완성을 기다리지 말 것 — 진짜 병목은 코드가 아니라
> 신뢰 증명 + 실제 제안이다. 목표: 실명 30명 → 5명 직접 메시지 → **첫 상담 1건.**

## 철칙 (메시지에 반드시 지킬 것)
1. **가격을 넣지 않는다.** 앵커 조기 노출·싸구려 인식 방지. 가격은 20분 콜에서만.
2. **증거로 연다.** "이미 실사용 서비스 3개를 만들었다" — 실력이 아니라 *제안*이 격차다.
3. **구체적 오퍼.** "2주 단위로 범위를 고정해서 진행" (막연한 "개발해줌" 금지).
4. **저부담 요청.** 계약이 아니라 "20분 통화 가능?"가 first ask.
5. **타깃.** 멋사 비개발 창업팀 / 해커톤·아이디어톤 / 창업동아리 / 교수·선배.

## 메시지 템플릿 (관계별 — 복붙 후 한 줄만 개인화)

### ① 가까운 멋사 동료·팀 (반말/캐주얼)
> 혹시 주변에 "아이디어는 있는데 만들 사람이 없는" 팀 있어?
> 나 이미 실사용 중인 서비스 3개 만들었고, 요즘 2주 단위로 범위 딱 고정해서
> MVP 만들어주는 거 시작하려고. 어떤 거 만들고 싶은지 20분만 통화해볼 수 있을까?

### ② 소개 요청 (1-hop — 본인 말고 주변 연결)
> {이름}아, 부탁 하나. 혹시 네 주변에 웹/앱 MVP 필요한데 개발자 못 구한 팀 있으면
> 나 연결해줄 수 있어? 실사용 서비스 3개 만든 거 있고, 2주 범위 고정으로 진행해.
> 가볍게 20분 통화부터 하면 돼. 한 명이라도 떠오르면 고맙겠다!

### ③ 교수·선배 (존댓말/격식)
> 교수님(선배님), 안녕하세요 김종건입니다.
> 멋사 풀스택 파트장 하면서 실사용 서비스 3개(ClassFileAuto·likelion-sch.com 등)를
> 직접 만들어 운영해 왔습니다. 요즘 MVP가 필요한 팀을 2주 단위 고정 범위로 돕는 일을
> 시작하려는데, 혹시 그런 팀이 주변에 있으면 소개 부탁드려도 될까요? 20분 통화면 충분합니다.

### ④ 단체 채팅 (해커톤·동아리 공지형)
> [개발 도와드려요] 아이디어 있는데 만들 사람 없는 팀 찾습니다.
> 실사용 서비스 3개 만든 풀스택이고, 2주 단위로 범위 고정해서 MVP 만들어드려요.
> 관심 있으면 DM 주세요 — 어떤 거 만들고 싶은지 20분 통화부터 가볍게!

## 퍼널 추적 (성공 기준)
- **1차 게이트(학습 신호):** 직접 연락 30명 → **상담 3건.** 미달이면 메시지/타깃/오퍼 수정(피벗 트리거).
- **2차(이번 달):** 상담 10건 → 유료 제안 3건 → **선금 받은 계약 1건.**

| 단계 | 목표 | 현재 |
|------|------|------|
| 리스트업(이름) | 30 | 0 |
| 메시지 발송 | 5+ | 0 |
| 상담(20분 콜) | 3 | 0 |
| 유료 제안 | 3 | 0 |
| 선금 계약 | 1 | 0 |

> 실명 30명 리스트는 **`outreach-contacts.local.md`** (gitignore됨)에 개인정보로 보관.
> 분류 칼럼 예: 이름 / 관계 / 타깃유형(멋사·해커톤·동아리·교수) / 우선순위 / 발송일 / 상태 / 메모.
